# 第 2 章 基本数据类型

**对象** 是可以在其中表示值的一块存储区域。C 标准将对象定义为：**执行环境中的数据存储区域，其内容可以表示值**。此外，对象在被引用时，对象可以具有特定的类型。

在 [第 1 章 快速入门](001-快速入门.md) 我们介绍过的 **变量** 就是对象的一个例子。在 C 语言中，每一个变量都有一个声明的 **类型**，该类型告诉编译器该变量的值代表的对象种类。例如，`int` 类型的对象包含的是整数值

> [!tip]
> 
> 类型的重要之处在于，如果将某种对象类型的位模式按照另一种对象类型来解释，则可能产生不同的值
> 
> 例如，数字 $1$ 在 IEEE 754 种的位模式是 $0x3f800000$ 表示。如果将这个位模式按照 `int` 类型来解释，得到的数值就是 $1065353216$。至于原因，我们将在本章后续内容介绍

计算机能够处理的最基本的数据就是数字；最擅长处理便是整数。本章我们主要介绍 C 语言中支持的基本数据类型，即 **数字**。其他任何类型都是建立在数字类型类型之上的。计算机中数字采用 **位模式**，由 $0$ 和 $1$ 组成的序列，也称为 **位序列**；也就是说，$0$ 和 $1$ 也称为 **位**

> [!tip]
> 
> 计算机世界中所有的数据都是由 $0$ 和 $1$ 组成的位序列以及按照某种 **解释** 形成的。解释定义了 **位序列的长度** 以及如何将位序列 **映射** 到现实世界中的数据
>
