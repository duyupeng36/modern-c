# 快速入门

学习 C 语言编程的最佳方法就是亲自动手写代码，而传统的入门程序就是在屏幕上输出 `"hello, world!"`。下面给出了示例程序 [hello.c](../src/ch01/hello.c) 的完整代码

```c title="hello.c"
/* hello.c - 向终端输出 hello, world */

#include <stdio.h>
#include <stdlib.h>

int main(int argc, [[maybe_unused]] char *argv[argc + 1]) {
    // 
    printf("hello, world!\n");
    return EXIT_SUCCESS;
}
```

> [!tip]
> 
> 注意，我们将示例程序保存为 `hello.c` 文件，文件扩展名 `.c` 表示该文件中包含的是 C 语言源代码
> 

稍后我们详细介绍这段代码的每一行内容，现在我们来看如何编译并运行它。要编译这个程序需要一个编译器。针对不同的操作系统，可以有不同的选择，下表列出了常用操作系统上的可选编译器或IDE

| 系统      | 编译器                                                                                                                                                                                      |
| :------ | :--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Windows | [GCC](https://gcc.gnu.org/) [Clang+LLVM](https://clang.llvm.org/)  [visual studio](https://visualstudio.microsoft.com/) [qt creator](https://www.qt.io/development/tools/qt-creator-ide) |
| Linux   | [GCC](https://gcc.gnu.org/) [Clang+LLVM](https://clang.llvm.org/)  [qt creator](https://www.qt.io/development/tools/qt-creator-ide)                                                      |
| MacOS   | [GCC](https://gcc.gnu.org/) [Clang+LLVM](https://clang.llvm.org/) [qt creator](https://www.qt.io/development/tools/qt-creator-ide)                                                       |

在 Winodws 系统上如何安装 GCC？。首先，访问 [WinLibs](https://winlibs.com/) 找到 `UCRT runtime` 选择最新的，目前是 [winlibs-x86_64-posix-seh-gcc-15.2.0-mingw-w64ucrt-14.0.0-r7.7z](https://github.com/brechtsanders/winlibs_mingw/releases/download/15.2.0posix-14.0.0-ucrt-r7/winlibs-x86_64-posix-seh-gcc-15.2.0-mingw-w64ucrt-14.0.0-r7.7z)。然后，下载并解压。最后，将解压目录下的 `bin` 目录添加到系统  `PATH`  环境变量中即可。这里我们只是安装了 GCC 编译器，还缺少一个编辑器用于编写 C 代码。在 Windows 系统中，一个很好的选择就是 [visual studio code](https://code.visualstudio.com/)。安装 vscode 非常简单，不再多讲

在 Linux 系统上如何安装 GCC？Linux 系统一般都会自带一个编译器，通常是 GCC。如果没有可以使用包管理工具进行安装。根据我们使用的 Linux 发行版不同，包管理工具也不同。

```shell
# Ubuntu
sudo apt install build-essential
# Archlinx
sudo pacman -S base-devel gdb
```

现在，我们已经安装好了 GCC 编译器。接下来就是运行编译器编译我们的 [hello](../src/ch01/hello.c) 程序。例如

```shell
# 使用 gcc 进行编译我们的 hello.c 程序
gcc -std=c23 -Wall -o hello hello.c

```

`gcc` 就是我们安装的编译器名字。选项 `-Wall` 告诉编译器为我们提供任何它认为所有不寻常的告警。选项 `-o hello` 告诉编译器将输出存储在名为 `hello` 的文件中。`hello.c` 是 C 源代码文件的名字
