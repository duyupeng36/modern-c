# Modern-C

C 语言自 20 世纪 70 年代由 [Ken Thompson](https://en.wikipedia.org/wiki/Ken_Thompson) 和 [Dennis Ritchie](https://en.wikipedia.org/wiki/Dennis_Ritchie) 等人在开发 [Unix](https://en.wikipedia.org/wiki/Unix) 操作系统时发明的程序设计语言初便已诞生。此后，C 语言被应用于各类项目中。用 C 编写的程序和系统无处不在，任何具有可编程接口的现代设备中都能看到它的身影

现如今的的 C 语言已不再是最初设计的那样了。早期的 C 被称为 **K&R C**（Kernighan and Ritchie C）。此后，C 经历了一个重要的标准化和扩展过程。目前 C 语言扩展由 **国际标准化组织**(International Organization for Standardization, ISO)推动。下表列出了 ISO 发布的几个重要的 C 标准

| 年份        | 标准                                      | 简称      |
|:----------|:----------------------------------------|:--------|
| 1989/1990 | (ANSI 标准 X3.159-1989) ISO/IEC 9899:1990 | C89/C90 |
| 1999      | ISO/IEC 9899:1999                       | C99     |
| 2011      | ISO/IEC 9899:2011                       | C11     |
| 2018      | ISO/IEC 9899:2018                       | C17     |
| 2024      | ISO/IEC 9899:2024                       | C23     |

> [!tip]
>
> C 标准委员会投入了大量精力来保证向后兼容性，即使是使用早期 C 标准编写的代码，在支持更新标准的编译器上应能编译为语义等价的可执行文件

下面我们将正式开启现代 C 语言的学习，学习的内容将涵盖 C89 至 C23 所有标准
