# 第六章 定义函数

将程序分解为易于管理的代码块是每种编程语言的基本理念。函数是 C 程序种的基本构建块。本质上就是一个命名的代码块。目前，我们大部分时间都在编写的函数是 `main()` 函数，并少数几个标准库种的函数。现在我们介绍如何自定义函数

## 6.1 程序的分解

前面我们编写的大部分程序都只有一个 `main()` 函数。实际上，C 程序是由无数个函数组成，每个函数都有明确定义的功能。C 程序从 `main()` 函数开始执行，通过 `main()` 函数 **调用** 其他函数，其他函数再调用别的函数

一个函数调用另一个函数，另一个函数再调用别的函数，此时，就会有好几个函数同时执行，每个函数都有另一个还未返回的调用函数，都在等待它调用的函数返回。显然，必须跟踪再内存的哪个地方进行函数调用，函数返回时从哪个地方继续执行。

这些跟踪函数状态的信息全部保存在 **栈** 上，它通常被称为函数的 **调用栈**。调用栈记录了所有函数调用的信息，以及传递给每个函数的详细信息

## 6.2 定义函数

函数时具有特定用途的自包含代码块。函数定义一般具有与 `main()` 函数相同的基本结构。函数定义由 **函数头** 和其后的一个包含函数代码的代码块组成，这个包含函数代码的代码块也称为 **函数体**。函数头一般需要指定三个方面的内容

+ 返回类型，即函数执行完毕时返回的值的类型
+ 函数名。函数根据与变量相同的规则命名
+ 调用函数时，可以传递给函数的数据项的个数和类型。这称为 **参数列表**，放在函数名后面的圆括号中，是一个用逗号分隔的列表

函数的一般表示形式如下

```c
T name(parameters)
{
    // code for the function
}
```

下面是一个函数定义的例子。它实现了基本的数学乘幂或求指数操作。

```c
double power(double x, double n)
{
    double result = 1.0;
    if ( n >= 0)
    {
        for (int i = 1; i <= n; ++i)
        {
            result *= x;
        }
    }
    else
    {
        for (int i = 1; i <= -n; ++i)
        {
            result /= x;
        }
    }
    return result;
}
```

在定义函数时，如果指定参数列表，那么在调用函数时，就需要按顺序提供对应的类型的参数，这个参数称为 **实际参数**；而定义函数时指定的参数列表称为 **形式参数**。`power()` 函数有两个形式参数 `x` 和 `n`。参数名在函数体种用于访问传递给函数的对应值。函数 `power()` 可以在程序的任何其他地方调用。调用函数时，需要使用函数名并跟随一个参数列表。例如

```c
double number = 3.0;
double result = power(number, 2); 
```

只有在调用 `power()` 函数时，函数体中的代码才会执行，并将形式参数 `x` 和 `n` 分别初始化为 `3` 和 `2`

C 语言编译器通过函数名唯一确定需要调用的函数，此时必须提供函数需要的 **实际参数**

> [!tip]
>
> 再次明确，术语 **实际参数**，简称 **实参**，表示在函数调用中传递给形式参数的值。
>
> 并且，函数调用中的实参顺序必须对应与函数定义中参数列表里的参数顺序
> 
> 具体来说，实参的数据类型对应与参数列表的数据类型。如果不能完全匹配，编译器会在合适的时候进行隐式类型转换
>
> 如果实际参数的类型无法通过隐式类型转换匹配参数类型，则编译将失败。
