# 现代 C 语言

1972年，Dennis Ritchie 和 Ken Thompson 在 贝尔电话实验室开发了 C 语言。C 语言的发明是为了编写名为 UNIX 的操作系统。UNIX 操作系统完全是用 C 语言编写的。

根据 TIOBE 编程社区指数，自 2001 年以来，C 语言一直位居最受欢迎编程语言排行榜的第一名或第二名。C 语言还是 TIOBE 2019 年的年度编程语言。C 语言如此流行的原因可能包含如下几点

1. **信任程序员**：C 语言假定程序员知道自己在做什么并且允许去做
2. **不阻止程序员做该做的事**：C 语言作为系统编程语言，必须能够处理各种底层任务
3. **保持语言自身的小巧和简单**：C 语言的设计旨在紧密贴近硬件，占用少量的空间
4. **一种操作只提供一种方法**：这也被称为 **机制守恒**，限制重复机制的引入
5. **要快**：即便无法保证可移植性，也要保证程序运行性能
