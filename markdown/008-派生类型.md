# 第 8 章 派生类型

## 8.1 类型定义

类型定义是创建派生类型的一种方式。在 C 语言中，可以使用 `typedef` 关键字定义类型

> [!tip]
>
> 注意，`typedef` 创建的只是类型 **别名**，并不会创建新类型

例如，下面的每个声明都创建了一个新的类型别名。

```c
typedef unsigned int uint_t;
typedef signed char schar_t, *schar_pt;
typedef signed char (*fp)(void);
```

在标准头文件中，以 ` _t` 结尾的标识符都是类型定义（现有类型的别名）。一般而言，在你自己的代码中，不要沿用这种命名惯例，因为 C 标准保留了符合模式 `int[0-9a-z_]*_t` 和 `uint[0-9a-z_]*_t` 的标识符

## 8.2 结构

结构类型(`struct`) 包含顺序分配的 **成员** 对象。每个对象都有自己的名称，类型可能并不一致。这一点儿不像数组，后者要求所有元素的类型必须相同

> [!tip]
>
> 换句话说，结构是一个或多个变量的集合，这些变量可能为不同的类型。结构将一组相关的对象看作一个单元而不是独立的实体，因此结构有助于组织复杂的数据

用于描述结构的一个典型的例子来自于图形领域：点由一对坐标定义，矩形由两个点定义，等等。其中一个点可以表示为一个结构，矩形是由两点组成的结构。

> [!tip]
>
> 从 C89 标准开始，结构是可以拷贝、复制、传递给函数，函数也可以将结构作为返回值。

### 8.2.1 结构变量

在图形领域中，点是最基本的对象。假设用 `x` 与 `y` 坐标表示，并且 `x` 和 `y` 的坐标值都是整数。我们可以采用结构存储这两个坐标，其声明如下

```c
struct point {
    int x;
    int y;
};
```

关键字 `struct` 引入结构声明。结构声明由包含在花括号内的一系列声明组成。关键字 `struct` 后面的名字是可选的，称为 **结构标记**，例如这里是 `point`

> [!tip]
>
> 结构标记用于为结构命名，在定义之后，**结构标记就代表花括号内的声明**，可以用它作为该声明的简写形式

结构中定义的变量称为 **成员**。编译器根据上下文可以识别标识符是结构成员、结构标记还是普通变量。因此，它们可以采用相同的名字。

**`struct {...}` 声明定义了一种数据类型**。在标志结构成员表结束的右花括号之后可以跟一个变量表，这与其它基本类型的变量声明是相同的

```c
struct { int x; int y;} point1, point2;
```

从语法角度来说，这种声明方式与普通变量声明是一样。这里的声明 `point1` 和 `point2` 都是 `struct { int x; int y;}` 类型的变量，并为它们分配存储空间

> [!tip]
>
> 如果结构声明的后面不带变量表，则不需要为它分配存储空间，它仅仅描述了一个结构的模板或轮廓

如果结构声明中带有标记，那么在以后定义结构变量时便可以使用该标记定义。例如，对于上述给出的结构 `point`，可以声明如下变量

```c
struct point p1;
```

结构的初始化与数组类似，在结构变量定义的后面使用初始化列表进行。初始化列表是 **使用花括号把要存储在结构变量中的数据包围起来**。需要注意的是，**初始化器中的值必须按照结构成员的顺序来显示**

```c
struct point p1 = {20, 30};
```

> [!warning]
>
> 如果初始化器中的成员数少于结构的成员，剩余的成员均被初始化为 `0`

从 C99 标准开始，指示器语法也适用于结构的初始化，指示器指定成员进行初始化。**不必记住原始声明时成员的顺序**，并且成员的顺序在之后发生了改变，也不会影响指示器语法。例如

```c
struct point p1 = {.y=20, .x=30};
```

在表达式中，可以通过下列形式引用某个特定结构中的成员

```c
结构名.成员名
```

其中成员运算符 `.` 将结构名与成员名连接起来。例如，可以使用下列语句打印店的坐标

```c
printf("(%d, %d)", p1.x, p1.y);
```

甚至可以通过下列语句计算点到原点 `(0, 0)` 的距离

```c
double dist = sqrt((double)p1.x * p1.x + (double)p1.y * p1.y);
```

结构可以嵌套，我们可以用对角线上的两个点来定义矩形，相应的结构定义如下

```c
struct rect {
    struct point p1;
    struct point p2;
};
```

结构 `rect` 包含两个 `point` 类型的成员。按照下列方式声明 `screen` 变量

```c
struct rect screen = {.p1={.y=10, .x=2}, .p2={.x=3, .y=7}};
```

此时，可以使用下列语句访问点 `p1` 的 `x` 坐标

```c
screen.p1.x
```

下面的示例程序 [`ex08_01.c`](../src/ch08/ex08_01.c) 中演示了关于结构体变量的使用

```c
/**
 * ex08_01.c - 结构体变量
**/

#include <stdio.h>
#include <math.h>

struct  point
{
    int x;
    int y;
};

struct rectangle
{
    struct point top_left;
    struct point bottom_right;
};

int main(void)
{
    struct point p1 = {0, 0};  // 顺序
    struct point p2 = {.y=10, .x=20};  // 指示器

    printf("p1: (%d, %d)\n", p1.x, p1.y);
    printf("p2: (%d, %d)\n", p2.x, p2.y);

    double distance = sqrtl(powl(p2.x - p1.x, 2) + powl(p2.y - p1.y, 2));
    printf("Distance: %.2f\n", distance);

    struct rectangle rect = {{.x=0, .y=0}, {.y=20, .x=10}};
    printf("Rectangle top-left: (%d, %d)\n", rect.top_left.x, rect.top_left.y);
    printf("Rectangle bottom-right: (%d, %d)\n", rect.bottom_right.x, rect.bottom_right.y);

    printf("Rectangle width: %d\n", rect.bottom_right.x - rect.top_left.x);
    printf("Rectangle height: %d\n", rect.bottom_right.y - rect.top_left.y);

    return 0;
}
```

### 8.2.2 结构与函数

结构的合法操作只有 **复制和赋值**、**取地址**、**访问成员**。请注意，**结构之间不能进行比较**

其中，复制和赋值包括向函数传递参数以及从函数返回、使用结构初始化另一个结构、使用结构向另一个结构赋值。向函数传递结果至少有三种方法：分别传递结构成员、传递整个结构、传递结构指针

首先，我们来看 `makepoint()` 函数，它需要两个 `int` 类型的参数，并返回一个 `struct point` 类型的值

```c
struct point makepoint(int x, int y)
{
    struct point point = {x, y};
    return point;
};
```

> [!tip]
>
> 注意，参数名和结构成员同名不会引起冲突。事实上，使用重名可以强调两者之间的关系
>

现在可以使用 `makepoint()` 函数动态地初始化任意结构，也可以向函数提供结构类型的参数

```c
struct rectangle screen;
screen.p1 = makepoint(0, 0);
screen.p2 = makepoint(10, 10);
struct point middle = makepoint((screen.p1.x + screen.p2.x) / 2, (screen.p1.y + screen.p2.y) / 2);

printf("middle point: (%d, %d)\n", middle.x, middle.y);
```

接下来需要编写一系列的函数对点执行算术运算。例如

```c
struct point addpoint(struct point p1, struct point p2)
{
    p1.x += p2.x;
    p1.y += p2.y;
    return p1;
};
```

其中，函数的参数和返回值都是结构类型。之所以直接将相加所得的结果赋值给 `p1`，而没有使用显式的临时变量存储，是为了强调结构类型的参数和其它类型的参数一样，都是通过值传递的

> [!tip]
>
> **结构类型的参数是通过值传递的**，与其他类型的变量是一样的

下面来看另外一个例子。函数 `prinrect` 判断一个点是否在给定的矩形内部。我们采用这样一个约定：矩形包括其左侧边和底边，但不包括顶边和右侧边

```c
int prinrect(struct point p, struct rect r)
{
    return p.x >= r.p1.x && p.x < r.p2.x && p.y >= r.p1.y && p.y < r.p2.y;
}
```
